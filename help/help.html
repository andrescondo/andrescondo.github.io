¿Nunca has descagado un archivo desde GITHUB?

No te preocupes, aquí te doy los pasos para que lo hagas rapido y sin demoras

1.- Tienes que buscar el botón verde como se puede ver en la imagen.

Si no tienes experiencia no te preocupes, no es díficil, solo tienes que pulsar el boton de [Descargar ZIP] o [Download ZIP](Si no tienes traducido tu navegador).

Si eres alguien con experiencia, puedes clonar el repositorio usando el enlace HTTP, o si usas llaves de seguidad puedes usar el enlace SSH.

NOTA: El diseño puede variar ya que Github esta actualizando el diseño, pero la funcionalidad es la misma
